JSON

JSON stands for JavaScript Object Notation. We'll use JSON to represent data that can then be rendered dynamically on a browser. The script below declares 2 obect varibles representing 2 people. Each has attributes first and last. Variable alice has additional attribute salary. 

<script>
  var alice = {
    first: "Alice",
    last: "Wonderland",
    salary: 100000
  }
  var bob = {
    first: "Bob",
    last: "Marley"
  }
</script>
