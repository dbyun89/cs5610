Link Applications

Do not declare scripts within HTML documents. Instead declare them in separate files and load them using the the src attribute of the script element.

Don't Litter the Name Space

Wrap your code in a immediately-invoked function expression, e.g., (function(){ ... })(); 

    (function()
    {

        ...put your code here...

    })();

Log Messages to the Console

Use console.log() to write messages to the browser's console. This is useful to debug your JavaScript programs
