Run jQuery Program On Document Load

jQuery can manipulate the DOM, but it needs to wait for the browser to parse the HTML text and convert it into a DOM. Use the jQuery() or $() function to be notified when the document is done loading. Pass a function that will be invoked when the DOM is ready. The code below uses the $() function to invoke function main() when the DOM is ready. The function can be called anything you want, e.g., init(), start(). 

  $(main);

  function main()
  {
    ... put your jQuery program here ...
  }
